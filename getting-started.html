Getting Started

Installation

To install Syberia, open up an R console and type the following: 

devtools::install_github("robertzk/syberia@install")
syberia.io::install_syberia()
library(syberia)

Hello World!

Let's start with an example model. 

syberia_create("myproject")
run("example")

Creating the Model File

Now we can start creating a model! Let's try to predict the chances of surviving the sinking of the RMS Titanic, via the Kaggle dataset. 

dir.create("models/dev/titanic")
