Syberia
Build predictive models faster than ever before.

Syberia is a framework for quickly trying different methods in data science. Syberia supports building, debugging, testing, and deploying classifiers developed in R.

Syberia includes support for importing data, preparing data, multiple statistical modeling methodologies, exporting trained classifiers, validating the results, and deploying as a REST service.

Getting Started

Syberia files are created via a Yeoman generator. To install Syberia, install Yeoman and the generator. 

brew install npm
npm install -g yo
npm install -g generator-syberia

Now we can start creating a model! Let's try to predict the chances of surviving the sinking of the RMS Titanic, via the Kaggle dataset. 

mkdir titanic
cd titanic
yo syberia

To predict, we have to make a model. We can do that with Yeoman. 

yo syberia:model
